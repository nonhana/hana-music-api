| 接口地址 | `/comment/dj`  |
| 请求方式 | `GET` / `POST` |

## 请求参数

| 参数     | 类型             | 必填 | 默认值 | 说明                                                                                  |
| -------- | ---------------- | :--: | ------ | ------------------------------------------------------------------------------------- |
| `id`     | string           |  ✅  | -      | 电台节目的 id                                                                         |
| `limit`  | number \| string |  —   | 20     | 取出评论数量 , 默认为 20                                                              |
| `offset` | number \| string |  —   | -      | 偏移数量 , 用于分页 , 如 :( 评论页数 -1)\*20, 其中 20 为 limit 的值                   |
| `before` | number \| string |  —   | -      | 分页参数,取上一页最后一项的 `time` 获取下一页数据(获取超过 5000 条评论的时候需要用到) |
